## 技术架构

### 意图识别 (IntentType)

```python
class IntentType(Enum):
    CHAT = "chat"           # 普通聊天
    STORE = "store"         # 存储记忆
    RETRIEVE = "retrieve"   # 检索记忆
    DELETE = "delete"       # 删除记忆
    EXECUTE_TASK = "execute_task"  # 【新增】执行任务/设置提醒
```

### 任务调度器 (TaskScheduler)

基于 APScheduler 实现，支持：

1. **一次性提醒** - 使用 `DateTrigger`
2. **周期性任务** - 使用 `CronTrigger`

```
用户输入 "明天下午3点提醒我看论文"
    ↓
意图识别 → EXECUTE_TASK
    ↓
解析时间 → 2026-03-13 15:00
    ↓
提取内容 → "看论文"
    ↓
创建任务 → ScheduledTask
    ↓
调度器注册 → APScheduler
    ↓
存储记忆 → SQLite
    ↓
返回确认 → "已为您设置提醒..."
    ↓
(时间到达)
    ↓
触发回调 → WebSocket通知 → 前端弹窗
```

### 数据库表 (scheduled_tasks)

```sql
CREATE TABLE scheduled_tasks (
    task_id TEXT PRIMARY KEY,
    user_id TEXT NOT NULL,
    task_type TEXT NOT NULL,        -- reminder/task/meeting/recurring
    content TEXT NOT NULL,
    title TEXT,
    description TEXT,
    scheduled_time TIMESTAMP,
    timezone TEXT DEFAULT 'Asia/Shanghai',
    is_recurring BOOLEAN DEFAULT 0,
    cron_expression TEXT,
    metadata TEXT,
    status TEXT DEFAULT 'pending',
    created_at TIMESTAMP,
    last_run_at TIMESTAMP,
    next_run_time TIMESTAMP,
    error_message TEXT
);
```

## API 接口

### 1. 创建任务

```http
POST /api/tasks
{
  "user_id": "user123",
  "content": "记得看论文",
  "reminder_time": "明天下午3点",  // 支持自然语言
  "title": "论文提醒"
}
```

### 2. 获取任务列表

```http
GET /api/tasks/{user_id}?status=scheduled&limit=50
```

### 3. 取消任务

```http
DELETE /api/tasks/{task_id}?user_id=user123
```

### 4. WebSocket 连接 (实时通知)

```javascript
const ws = new WebSocket(`ws://localhost:8000/ws/${userId}`);
ws.onmessage = (event) => {
    const data = JSON.parse(event.data);
    if (data.type === 'reminder') {
        showNotification(data.title, data.content);
    }
};
```

## 与 AstrBot 的对比

| 功能 | AstrBot | 智忆助理 |
|------|---------|----------|
| 调度器 | APScheduler | APScheduler |
| 任务存储 | SQLite + SQLModel | SQLite |
| 触发通知 | 平台适配器(飞书/钉钉) | WebSocket + 前端弹窗 |
| 自然语言时间 | 支持 | 支持 |
| 周期性任务 | 支持 | 支持 |
| 任务API | RESTful API | RESTful API |
| 与记忆系统集成 | 独立存储 | 【优势】存储到记忆库，可检索历史提醒 |
