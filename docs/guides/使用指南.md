## 三、使用方法

### 方法1：通过对话设置提醒（最简单）

直接发送自然语言消息，系统会自动识别并创建提醒：

```bash
# 一次性提醒
curl -X POST "http://localhost:8000/api/chat" \
  -H "Content-Type: application/json" \
  -d '{
    "user_id": "user123",
    "message": "明天下午3点提醒我看论文"
  }'
```

**响应示例：**
```json
{
  "success": true,
  "data": "已为您设置提醒📅\n\n时间：明天下午3点\n内容：看论文\n\n到时间我会通过系统通知提醒您！"
}
```

**支持的表达：**
- "明天下午3点提醒我开会"
- "后天早上9点叫我起床"
- "每天早上9点提醒我吃维生素"
- "每周五下午提醒我交周报"

### 方法2：通过 API 直接创建任务

```bash
# 创建一次性提醒
curl -X POST "http://localhost:8000/api/tasks" \
  -H "Content-Type: application/json" \
  -d '{
    "user_id": "user123",
    "content": "记得看论文",
    "reminder_time": "明天下午3点",
    "title": "论文提醒"
  }'
```

**响应：**
```json
{
  "success": true,
  "task_id": "task_abc123",
  "scheduled_time": "2026-03-13T15:00:00",
  "message": "已设置提醒：明天下午3点 - 记得看论文"
}
```

```bash
# 获取任务列表
curl "http://localhost:8000/api/tasks/user123?status=scheduled"
```

**响应：**
```json
{
  "success": true,
  "data": [
    {
      "task_id": "task_abc123",
      "task_type": "reminder",
      "content": "记得看论文",
      "title": "论文提醒",
      "scheduled_time": "2026-03-13T15:00:00",
      "status": "scheduled",
      "is_recurring": false
    }
  ],
  "total": 1
}
```

```bash
# 取消任务
curl -X DELETE "http://localhost:8000/api/tasks/task_abc123?user_id=user123"
```

### 方法3：使用 Python 代码调用

```python
import requests
import json

API_URL = "http://localhost:8000"

# 设置提醒
def create_reminder(user_id, content, time):
    response = requests.post(f"{API_URL}/api/tasks", json={
        "user_id": user_id,
        "content": content,
        "reminder_time": time,  # 支持自然语言
        "title": "提醒"
    })
    return response.json()

# 获取任务列表
def get_tasks(user_id):
    response = requests.get(f"{API_URL}/api/tasks/{user_id}")
    return response.json()

# 使用示例
result = create_reminder("user123", "看论文", "明天下午3点")
print(result)
# {'success': True, 'message': '已设置提醒：明天下午3点 - 看论文'}
```

## 四、前端接收实时提醒（WebSocket）

### JavaScript 示例

```javascript
// 用户ID
const userId = 'user123';

// 1. 建立 WebSocket 连接
const ws = new WebSocket(`ws://localhost:8000/ws/${userId}`);

// 2. 连接成功
ws.onopen = () => {
    console.log('已连接到提醒服务');

    // 发送心跳（保持连接）
    setInterval(() => {
        if (ws.readyState === WebSocket.OPEN) {
            ws.send('ping');
        }
    }, 30000);
};

// 3. 接收消息（重点是这里！）
ws.onmessage = (event) => {
    const data = JSON.parse(event.data);

    // 收到提醒！
    if (data.type === 'reminder') {
        console.log('收到提醒:', data);

        // 显示浏览器通知
        if (Notification.permission === 'granted') {
            new Notification(data.title, {
                body: data.content,
                icon: '/notification-icon.png'
            });
        }

        // 或者显示自定义弹窗
        alert(`⏰ ${data.title}\n${data.content}`);

        // 播放提醒音
        const audio = new Audio('/notification-sound.mp3');
        audio.play().catch(e => console.log('播放失败:', e));
    }
};

// 4. 连接断开
ws.onclose = () => {
    console.log('连接已断开');
};

// 5. 请求浏览器通知权限
if ('Notification' in window && Notification.permission === 'default') {
    Notification.requestPermission();
}
```

### Vue3 组件完整示例

```vue
<template>
  <div class="chat-container">
    <!-- 消息列表 -->
    <div class="messages">
      <div v-for="msg in messages" :key="msg.id"
           :class="['message', msg.role]">
        {{ msg.content }}
      </div>
    </div>

    <!-- 输入框 -->
    <div class="input-area">
      <input v-model="inputMessage"
             @keyup.enter="sendMessage"
             placeholder="输入消息...比如'明天下午3点提醒我看论文'" />
      <button @click="sendMessage">发送</button>
    </div>

    <!-- 提醒弹窗 -->
    <div v-if="currentReminder" class="reminder-modal">
      <div class="reminder-content">
        <h3>⏰ {{ currentReminder.title }}</h3>
        <p>{{ currentReminder.content }}</p>
        <button @click="dismissReminder">知道了</button>
      </div>
    </div>
  </div>
</template>

<script setup>
import { ref, onMounted, onUnmounted } from 'vue';

const userId = 'user123';
const messages = ref([]);
const inputMessage = ref('');
const currentReminder = ref(null);
let ws = null;

// 连接 WebSocket
onMounted(() => {
  ws = new WebSocket(`ws://localhost:8000/ws/${userId}`);

  ws.onmessage = (event) => {
    const data = JSON.parse(event.data);

    if (data.type === 'reminder') {
      // 显示提醒弹窗
      currentReminder.value = data;

      // 播放提醒音
      new Audio('/notification-sound.mp3').play().catch(() => {});
    }
  };

  // 心跳
  const heartbeat = setInterval(() => {
    if (ws.readyState === WebSocket.OPEN) {
      ws.send('ping');
    }
  }, 30000);

  onUnmounted(() => {
    clearInterval(heartbeat);
    ws.close();
  });
});

// 发送消息
async function sendMessage() {
  if (!inputMessage.value.trim()) return;

  const message = inputMessage.value;
  messages.value.push({ id: Date.now(), role: 'user', content: message });
  inputMessage.value = '';

  const res = await fetch('/api/chat', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ user_id: userId, message })
  });

  const result = await res.json();
  messages.value.push({
    id: Date.now() + 1,
    role: 'assistant',
    content: result.data
  });
}

// 关闭提醒弹窗
function dismissReminder() {
  currentReminder.value = null;
}
</script>

<style>

  top: 0; left: 0; right: 0; bottom: 0;

}

```

## 五、完整工作流程演示

### 步骤1：设置提醒

```bash
# 用户发送消息
curl -X POST "http://localhost:8000/api/chat" \
  -H "Content-Type: application/json" \
  -d '{
    "user_id": "user123",
    "message": "1分钟后提醒我测试"
  }'
```

### 步骤2：系统处理

```
用户: "1分钟后提醒我测试"
  ↓
意图识别 → EXECUTE_TASK
  ↓
解析时间 → 当前时间 + 1分钟
  ↓
创建任务 → task_xxx (scheduled_time: 2026-03-12 14:30:00)
  ↓
APScheduler → 注册任务 (DateTrigger)
  ↓
SQLite → 保存到 scheduled_tasks 表
  ↓
回复用户 → "已为您设置提醒📅..."
```

### 步骤3：等待触发

```
(1分钟后)
  ↓
APScheduler → 触发任务
  ↓
回调函数 → WebSocket推送
  ↓
前端弹窗 → "⏰ 提醒 - 测试"
```

## 六、API 速查表

| 功能 | 方法 | 路径 | 参数 |
|------|------|------|------|
| 对话（自动识别提醒） | POST | /api/chat | user_id, message |
| 创建任务 | POST | /api/tasks | user_id, content, reminder_time, title |
| 获取任务列表 | GET | /api/tasks/{user_id} | status, limit |
| 取消任务 | DELETE | /api/tasks/{task_id} | user_id |
| WebSocket 连接 | WS | /ws/{user_id} | - |

## 七、常见问题

### Q1: 如何测试提醒是否有效？

使用"1分钟后"作为测试：
```bash
curl -X POST "http://localhost:8000/api/chat" \
  -H "Content-Type: application/json" \
  -d '{"user_id":"test","message":"1分钟后提醒我测试"}'
```

### Q2: 服务重启后任务会丢失吗？

不会。任务会保存在 SQLite 数据库的 `scheduled_tasks` 表中，重启后会自动恢复未完成的任务。

### Q3: 如何查看所有待执行的任务？

```bash
curl "http://localhost:8000/api/tasks/user123?status=scheduled"
```

### Q4: 支持哪些时间格式？

支持自然语言和 ISO 格式：
- "明天下午3点"
- "后天早上9点"
- "2026-03-15T15:00:00"
- "10分钟后"
