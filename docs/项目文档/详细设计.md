## 1. 核心类详细设计

### 1.1 MemoryMateAgent (系统入口)

```python
class MemoryMateAgent:
    """
    主代理类，协调各模块工作

    Attributes:
        memory_storage: MemoryStorage      # 记忆存储
        memory_manager: MemoryManager      # 记忆管理
        workflow_engine: MemoryWorkflowEngine  # 工作流引擎
        precise_scheduler: PreciseScheduler    # 精准调度器
        memory_crud: MemoryCRUD            # CRUD 操作
    """

    async def chat(self, user_id: str, message: str) -> str:
        """
        主对话接口

        Flow:
            1. 检查是否是确认回复
            2. 工作流引擎处理意图
            3. 根据意图执行对应操作
            4. 如果是提醒且无需确认，自动创建任务
        """

    async def _handle_confirmation(self, user_id: str, message: str) -> str:
        """
        处理用户的确认/取消回复

        用于时间冲突时的二次确认
        """
```

### 1.2 WorkflowEngine (意图识别)

```python
class MemoryWorkflowEngine:
    """
    工作流引擎：意图识别 + 流程编排

    IntentType 枚举:
        CHAT: 直接对话
        STORE: 存储记忆
        RETRIEVE: 检索记忆
        EXECUTE_TASK: 设置提醒
        DELETE: 删除记忆
    """

    async def process(self, user_id, message, current_time) -> WorkflowContext:
        """
        处理用户请求的主流程

        Steps:
            1. 意图识别 (_recognize_intent)
            2. 执行对应工作流
            3. 返回结果上下文
        """

    async def _recognize_intent(self, message: str) -> tuple[IntentType, str, str]:
        """
        识别用户意图

        Priority:
            1. 规则匹配（删除指令）
            2. 本地时间解析（相对时间）
            3. 大模型识别（兜底）
        """
```

### 1.3 PreciseScheduler (精准调度器)

```python
class PreciseScheduler:
    """
    精准定时调度器（替代 APScheduler）

    原理: 每个任务创建一个 asyncio.Task
          使用 asyncio.sleep 精准等待到执行时间

    Attributes:
        _tasks: Dict[str, PreciseTask]        # 任务字典
        _callbacks: List[Callable]            # 回调函数列表
        _async_tasks: Dict[str, asyncio.Task] # asyncio Task 对象
    """

    async def create_reminder(
        self,
        user_id: str,
        content: str,
        reminder_time: datetime,
        title: str = None,
        metadata: dict = None
    ) -> PreciseTask:
        """
        创建定时提醒

        Flow:
            1. 创建 PreciseTask 对象
            2. 保存到数据库
            3. 启动 asyncio Task 等待执行
            4. 返回 task 对象
        """

    async def _wait_and_execute(self, task: PreciseTask):
        """
        等待并执行任务

        计算 wait_seconds = (scheduled_time - now).total_seconds()
        使用 asyncio.sleep(wait_seconds) 精准等待
        支持任务取消（通过 asyncio.Event）
        """
```

```python
class MemoryStorage:

        存储记忆

        """
```

## 2. 数据库设计

### 2.1 SQLite 表结构

```sql
-- 记忆表
CREATE TABLE memories (
    id TEXT PRIMARY KEY,
    user_id TEXT NOT NULL,
    content TEXT NOT NULL,
    memory_type TEXT,  -- 'event', 'preference', 'attribute', 'reminder'
    event_date TEXT,
    event_time TEXT,
    importance INTEGER DEFAULT 1,
    access_count INTEGER DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 平台配置表
CREATE TABLE platform_configs (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id TEXT NOT NULL,
    platform_type TEXT NOT NULL,  -- 'lark', 'dingtalk', 'wechat'
    config TEXT NOT NULL,         -- JSON 格式
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    UNIQUE(user_id, platform_type)
);

-- 定时任务表
CREATE TABLE precise_tasks (
    task_id TEXT PRIMARY KEY,
    user_id TEXT NOT NULL,
    content TEXT NOT NULL,
    title TEXT,
    scheduled_time TIMESTAMP,
    status TEXT DEFAULT 'pending',  -- 'pending', 'completed', 'cancelled'
    metadata TEXT
);
```

## 5. 关键算法

### 5.1 混合检索排序

```python
def hybrid_rank(vector_results, bm25_results, alpha=0.6):
    """
    混合排序算法

    Args:
        vector_results: 向量检索结果 [(id, score), ...]
        bm25_results: BM25 检索结果 [(id, score), ...]
        alpha: 向量权重 (0-1)

    Returns:
        排序后的结果列表

    Algorithm:
        1. 归一化两种分数到 [0, 1]
        2. 计算加权得分: hybrid_score = alpha * vector_score + (1-alpha) * bm25_score
        3. 按 hybrid_score 降序排列
    """
```
