# 智忆助理 (MemoryMate) 概要设计

## 1. 项目定位

智忆助理是一款具备**长期记忆能力**的 AI 助手，能够：
- 存储和检索用户的个人信息、计划安排、喜好习惯
- 设置定时提醒并通过第三方平台（飞书）推送
- 基于历史记忆进行个性化对话

### 2.2 核心流程

#### 意图识别流程

```
用户输入
    ↓
规则匹配（删除指令）
    ↓
本地时间解析（time_parser）
    ↓
大模型意图识别（LLM）
    ↓
├─→ memory_storage: 存储记忆
├─→ memory_retrieval: 检索记忆
├─→ execute_task: 设置提醒
└─→ direct_chat: 直接回答
```

#### 定时提醒流程

```
用户: "2分钟后提醒我测试"
    ↓
意图识别: execute_task
    ↓
时间解析: 当前时间 + 2分钟
    ↓
创建任务: PreciseScheduler.create_reminder()
    ↓
异步等待: asyncio.sleep(120秒)
    ↓
时间到!
    ↓
回调触发: notify_user(task)
    ↓
├─→ WebSocket 推送（浏览器）
└─→ 飞书消息推送（LarkAdapter）
```

## 3. 四大核心功能

### 3.1 记忆存储 (memory_storage)

**触发条件**: 用户陈述个人信息、计划安排、喜好习惯

**示例**:
- "我喜欢喝浓缩咖啡" → 存储偏好
- "明天中午12点去吃麦当劳" → 存储日程
- "我3月15日要去上海" → 存储计划

**实现**:
- 文本存入 SQLite
- 向量存入 FAISS
- 时间信息结构化提取

### 3.2 记忆检索 (memory_retrieval)

**触发条件**: 用户询问之前记住的信息

**示例**:
- "我喜欢喝什么？"
- "我明天有什么安排？"
- "我上周做了什么？"

**实现**:
- 向量相似度检索
- BM25 关键词检索
- 混合排序返回结果

### 3.3 定时提醒 (execute_task)

**触发条件**: 用户明确要求"提醒"、"叫我"

**示例**:
- "明天下午3点提醒我开会"
- "2分钟后提醒我去吃饭"
- "每天早上9点提醒喝水"

**实现**:
- `PreciseScheduler`: 精准到秒的定时器
- 每个任务独立 asyncio.Task
- 支持取消、持久化

### 3.4 直接对话 (direct_chat)

**触发条件**: 闲聊、问候、常识问答

**示例**:
- "你好"
- "今天天气怎么样"
- "帮我写首诗"

**实现**:
- 直接调用 LLM 生成回复
- 不存储、不检索

## 4. 关键技术选型

| 技术 | 用途 |
|------|------|
| FastAPI | Web 框架，提供 REST API 和 WebSocket |
| Vue3 + Element Plus | 前端 UI 框架 |
| SQLite | 结构化数据存储 |
| FAISS | 向量索引，语义检索 |
| OpenAI API | 大模型对话、意图识别 |
| APScheduler | 传统任务调度（已废弃）|
| asyncio | 精准定时调度 |
| 飞书 API | 消息推送 |

## 5. 意图识别规则

### 5.1 判断优先级

1. **规则匹配**（最高优先级）
   - 删除指令: `^删除`, `^移除`

2. **本地时间解析**
   - "2分钟后"、"1小时后"

3. **大模型识别**（兜底）
   - 区分 "日程安排" vs "设置提醒"
   - 关键: 是否有"提醒"、"叫我"等关键词

### 5.2 关键区分

| 输入 | 意图 | 原因 |
|------|------|------|
| "明天中午12点去吃麦当劳" | memory_storage | 陈述计划，无"提醒"关键词 |
| "明天中午12点提醒我去吃麦当劳" | execute_task | 明确要求提醒 |
| "2分钟后提醒我看megtron" | execute_task | 明确说"提醒" |
