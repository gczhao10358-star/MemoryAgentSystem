# 智忆助理 (MemoryMate) 项目结构

## 目录树

```
记忆助理智能体/
├── api.py                      # FastAPI 后端服务入口
├── main.py                     # 命令行交互入口
├── config.yaml                 # 配置文件
├── requirements.txt            # Python 依赖
├── data/                       # 数据存储目录
│   ├── memory.db              # SQLite 数据库
│   ├── vector_index.faiss     # FAISS 向量索引
│   └── *_profile.json         # 用户画像数据
├── memory_assistant/           # 核心 Python 包
│   ├── core/                  # 核心逻辑模块
│   │   ├── memory_mate_agent.py    # 主代理类
│   │   ├── workflow_engine.py      # 工作流引擎（意图识别）
│   │   ├── memory_manager.py       # 记忆管理
│   │   ├── precise_scheduler.py    # 精准定时调度器
│   │   └── task_scheduler.py       # 传统任务调度器
│   ├── models/                # 数据模型
│   │   ├── memory.py          # 记忆模型
│   │   ├── user_profile.py    # 用户画像模型
│   │   └── retrieval.py       # 检索结果模型
│   ├── storage/               # 存储层
│   │   ├── memory_storage.py  # 记忆存储
│   │   ├── vector_store.py    # 向量存储
│   │   └── metadata_store.py  # 元数据存储
│   ├── retrieval/             # 检索模块
│   │   ├── hybrid_retrieval.py     # 混合检索
│   │   └── sparse_retrieval.py     # 稀疏检索
│   ├── profile/               # 画像模块
│   │   ├── profile_manager.py      # 画像管理
│   │   └── profile_learner.py      # 画像学习
│   ├── platform/              # 第三方平台
│   │   └── lark_adapter.py    # 飞书适配器
│   └── utils/                 # 工具类
│       ├── time_parser.py     # 时间解析
│       ├── llm_client.py      # LLM 客户端
│       └── embedding.py       # 嵌入模型
├── frontend/                   # 前端 Vue3 项目
│   ├── src/
│   │   ├── views/            # 页面组件
│   │   ├── stores/           # Pinia 状态管理
│   │   └── components/       # 通用组件
│   └── package.json
└── logs/                       # 日志目录
```

## 核心模块说明

### 1. Core 模块

| 文件 | 职责 |
|------|------|
| `memory_mate_agent.py` | 系统入口，协调各模块工作 |
| `workflow_engine.py` | 意图识别、工作流编排 |
| `precise_scheduler.py` | 精准定时提醒（asyncio）|
| `memory_manager.py` | 工作记忆、上下文管理 |

### 2. Storage 模块

| 文件 | 职责 |
|------|------|
| `memory_storage.py` | 记忆 CRUD 操作 |
| `vector_store.py` | FAISS 向量索引管理 |
| `metadata_store.py` | SQLite 数据库操作 |

### 3. Platform 模块

| 文件 | 职责 |
|------|------|
| `lark_adapter.py` | 飞书机器人消息推送 |

## 数据流

```
用户输入 → WorkflowEngine (意图识别)
    ↓
├─→ 存储记忆 → MemoryStorage → SQLite/FAISS
├─→ 检索记忆 ← HybridRetrieval ← VectorStore
├─→ 设置提醒 → PreciseScheduler → 定时触发 → LarkAdapter
└─→ 直接回答 → LLMClient → 回复用户
```
